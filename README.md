# Kanojo

This repository contains the main code for the Kanojo Plex Agent. This allows you to query Kanojo for movie and cast information through your Plex server.

**Note: This agent is still in alpha. It currently points to 127.0.0.1 as a URL for Kanojo.**

## Installation

Clone the repository into your [Plex Plug-ins folder](https://support.plex.tv/articles/201106098-how-do-i-find-the-plug-ins-folder/) and restart your Plex server.

When creating a library, you will need to enter a valid API Token in the Advanced settings of the library.
